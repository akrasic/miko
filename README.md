# Miko

Search for script installations under a cPanel account

Supported scripts ( so far ):
- Wordpress
- Joomla  1.x , 2.x, 3.x
- Drupal 7.x, 6.x
- Magento
- phpBB
- SMF Forum
- MyBB
- Moodle

## Installation

gem install miko

## Usage
``$ miko USERNAME``
